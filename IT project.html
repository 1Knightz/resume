project description

My project is about the development and use of cloud space. The reason I developed this project is to better protect those who need private communication or more important information transmission. system introductionn

In my system, the message is sent by the computer host, and then transmitted through the network to the cloud storage host, and finally filtered by the firewall, and the information is selected for entry.

Flowchart introduction

My process is to send according to the first device information, send it to the network of the computer, send it to the receiving receiver through the transmission of the computer terminal, and finally transmit it to the cloud space for storage through the filtering of the firewall.
